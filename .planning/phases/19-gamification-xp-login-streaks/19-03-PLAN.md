# Plan 03: Achievements Tab UI & Gamification Frontend Module

## Objective

Create the Achievements Tab UI within Profile Settings and implement the frontend gamification display module. Add HTML modals for Streak Splash and Morning Prompt. Build profile.js with data fetching, SVG progress circles, badge grid, and tab management.

Purpose: Frontend UI for gamification display. Allows users to see XP progress, streaks, and earned badges in a zen-minimalist interface.
Output: Achievements Tab in Profile, two SVG progress circles, badge grid, splash screens, gamification module.

## Tasks

### Task 1: Add Achievements Tab and Splash Modals to index.html

**File:** index.html

1. In modal-settings (Profile Settings modal), add a fourth tab button and pane for Achievements:
   - Tab button: data-target="tab-achievements"
   - Tab pane: id="tab-achievements" with class "profile-tab-pane hidden"
   
2. Add Achievements tab content:
   - Streak display section showing current streak number
   - Two SVG progress circles side-by-side (Daily and Overall)
   - Daily circle: #6B47F5 (purple), label "Daily"
   - Overall circle: #10B981 (mint), label "Overall"
   - Badge grid for earned badges (newest-first)

3. Add Streak Splash Modal:
   - id="modal-streak-splash", hidden initially
   - Shows flame icon, streak number, "Day Streak!" label, inspirational message
   - Auto-dismisses (handled by JS)

4. Add Morning Prompt Modal:
   - id="modal-morning-prompt", hidden initially
   - Shows "Yesterday's unfinished tasks" heading
   - Task list area (injected by JS)
   - "All done" button to close

**Verify:**
- HTML parses without errors
- Achievements tab button visible in Profile Settings
- SVG circles render without distortion (both visible, not nested)
- Colors correct: daily #6B47F5, overall #10B981
- Modals exist in DOM and are hidden initially

### Task 2: Implement frontend/js/profile.js

**File:** frontend/js/profile.js (new file)

Create as ES6 module with these exports:
- `initGamification()` — fetches summary and renders Achievements Tab
- `updateXPDisplay(xpResult)` — updates circles after XP award
- `showStreakSplash(streak, isMilestone)` — displays splash screen
- `showMorningPrompt(tasks)` — displays morning prompt modal
- `registerLoginCheckFlow()` — runs login-check and triggers splash/prompt

Core functions:
1. renderAchievementsTab(summary) — updates streak display, XP circles, badge grid
2. updateXPCircles(xpData) — calculates stroke-dashoffset for smooth progress animation
3. renderBadgeGrid(badges) — renders badges newest-first with icons
4. showStreakSplash — displays splash with milestone-specific messaging
5. showMorningPrompt — lists tasks with reschedule buttons
6. registerLoginCheckFlow — runs login-check POST and handles first-login flow

SVG circles formula:
- stroke-dasharray = 238.76 (circumference for r=38)
- stroke-dashoffset = 238.76 * (1 - progress_fraction)
- Animates on progress change with CSS transition

**Verify:**
- Module imports without syntax errors
- initGamification fetches summary and renders correctly
- updateXPCircles updates stroke-dashoffset smoothly
- Splash shows and auto-dismisses after 4 seconds
- Morning prompt lists tasks and handles reschedule buttons
- Badges render with icons and names
